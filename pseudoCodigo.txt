verificaPosicao  Função para verificar se a posição atual é valida, ou seja, se a posicao (i,j) pertence no tabuleiro n x m.

Função princial do passeio 
    caso base (primeiro if): Se o tamanho do caminho é igual a n X m chama a função que verifica se para todos os 8 possíveis próximos movimentos a nova posição é diferente da posição inicial: 
                    Se for: Sucesso, salva esse caminho
                    Se não: Falha, backtraking / lança uma lista vazia
    else: Chama a função que verifica limites
            Se retornar true (ou seja, posição dentro dos limites) pode seguir a recursão
            Se não deu falha, backtraking / lança uma lista vazia
    else 2: verifica se a posição já não está na lista
    coloca um otherwise com os 8 posssíveis movimentos:
        let 
           m1 = passeio(l-2, c-1)
           m2 = passeio(l-2, c+1)
           m3 = passeio(l-1, c-2)
           m4 = passeio(l-1, c+2)
           m5 = passeio(l+1, c-2)
           m6 = passeio(l+1, c+2)
           m7 = passeio(l+2, c-1)
           m8 = passeio(l-2, c+1)
        
        new_m1 = [ n : c | c <- m1 ]
        new_m2 = [ n : c | c <- m2 ]
        new_m3 = [ n : c | c <- m3 ]
        new_m4 = [ n : c | c <- m4 ]
        new_m5 = [ n : c | c <- m5 ]
        new_m6 = [ n : c | c <- m6 ]
        new_m7 = [ n : c | c <- m7 ]
        new_m8 = [ n : c | c <- m8 ]

    in new_m1 ++ new_m2 ++ new_m3 ++ new_m4 ++ new_m5 ++ new_m6 ++ new_m7 ++ new_m8

    movimentos = [(-2, -1), (-2, 1), (-1, -2), (-1, 2), (1, -2), (1, 2), (2, -1), (2, 1)]
    candidatos = [ (l + dr, c + dc) | (dr, dc) <- movimentos ]

saindo da função verfica se a lista está vazia (retorna que não foi possível encontrar caminhos) ou imprime todos caso não esteja
